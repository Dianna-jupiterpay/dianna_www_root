               Data Update Interval 
              

                This is the period of time between updates to the Dashboard panels.
                

                Note: Setting the update interval too low can slow your server's response
                  time.
                

                
              

            

            

               Auto Logout 
              

                Amount of time of inactivity after which the user is logged out.
                

                
              

            

            

               Daily Process Time 
              

                This is the time of day that a daily data prep routine will run. This should run
                at a time of day when the server is running and POS system usage is minimal or non-existent.
